# The Maze Game

The Maze Game is a simple React-based maze escape game. The player has to move through a fixed maze and reach the exit before the time runs out.

## Features

- Fixed maze map
- 40-second countdown timer
- Player movement using keyboard
- On-screen movement buttons
- Move counter
- Win and lose message
- Restart game option
- Responsive design for different screen sizes

## Game Rules

The player starts from the red circle. The exit point is shown in green. The player must reach the exit before the timer becomes zero.

If the player reaches the exit within 60 seconds, the player wins. If the timer reaches zero before reaching the exit, the player loses.

### Keyboard Controls

- Arrow Up or W = Move Up
- Arrow Down or S = Move Down
- Arrow Left or A = Move Left
- Arrow Right or D = Move Right


### Button Controls

The game also has on-screen buttons:

- Up
- Left
- Down
- Right
- Restart Game
